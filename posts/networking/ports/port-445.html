üîç What is Port 445?

  • Protocol: TCP
  • Used for: Microsoft SMB (Server Message Block) over TCP
  • Purpose: File sharing, printer sharing, remote access, network browsing, etc.

Since Windows 2000, SMB runs directly over TCP (port 445) without using NetBIOS (which used ports 137-139).

  1. SMB is Powerful
    2. SMB allows file sharing, remote command execution, and printer access ‚Äî perfect for attackers to move laterally in a network or exfiltrate data.
  2. Commonly Exposed
    3. Insecure systems often expose port 445 to the internet ‚Äî a huge risk.
  3. Vulnerabilities and Exploits
    • EternalBlue (MS17-010) ‚Äì Used in WannaCry, NotPetya ransomware attacks.
    • SMBGhost (CVE-2020-0796) ‚Äì RCE vulnerability in SMBv3 (Windows 10, Server 2019).
    • Relaying attacks ‚Äì NTLM hashes can be captured and relayed to authenticate without knowing passwords.

Attack Type Description
